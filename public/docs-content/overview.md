# HoneyOps technical documentation

HoneyOps is sdsa SaaS platform for deploying and operating honeypots. It ships as a lightweight `.ova`
virtual appliance that pairs with your cloud console using a one-time hook key and continuously
sends health and incident telemetry.

## What you'll find here

- Getting started guides
- Deployment and networking
- Console features
- API and integrations
- Troubleshooting
